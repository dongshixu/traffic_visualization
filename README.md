# Vue + Django
> "node.js >= 6.0.0" 

> "vue cli2"

## 1 安装
### (1) 安装node.js
### (2) 安装vue cli2
### (3) 安装 cnpm ，打开 cmd 并输入：（如果失败，请以管理员权限打开cmd）
> npm install -g cnpm --registry=https://registry.npm.taobao.org
### (4) 在 \frontend 中打开 cmd，并输入以下命令：
> cnpm install  // 安装第三方库 

> npm run dev  // 运行前端, 默认地址为:localhost:8080
